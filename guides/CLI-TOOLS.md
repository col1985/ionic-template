## CLI Tools

### commit

```bash
npm run commit
```

### local development

Run app locally in the browser. Linter is disabled in the wtahc config.

```bash
npm run serve
```

### lint

Manually lint codebase.

```bash
npm run lint
```

### docs

Generate source-code documentation using the Typedoc tool.

```bash
npm run docs
```

### tests

```bash
npm test
```

### clean

```bash
npm run ionic:build
```

### build

```bash
npm run build
```
